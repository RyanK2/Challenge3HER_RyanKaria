# Challenge3API
 
Combinatie tussen de Google Maps API & Openweather API. Als locatie op de Google Maps API heb ik een pin vastgezet aan de haagse hogeschool om aan te geven
dat dit de huidig gekozen plek is.
De openweather API zorgt ervoor dat je een stad kan opzoeken en hiervan de naam, temperatuur en weersverwachting te zien krijgt om zo de beste landingsplek 
in de nabije omgeving kan opzoeken.
De website maakt gebruik van 3 breakpoints en een grid.
